# ARSystem

環境設定

1, Podファイルを元に必要なフレームワークのインストール

pod install（初回）・・・Podfile.lockを生成（ファイルがあれば参照してインストール）

pod update（2回目以降）・・・Podfile.lockを参照せずにインストール，Podfile.lockを更新



2, OpenCVのインストール

下記のリンクからopencv-0.0.0-ios-framework.zipをインストール
